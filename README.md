# Task Management Application

This project is a Task Management Application developed using React and Chakra UI. Users can create, manage, and view the history of their tasks.

## Features:
- Task Creation
- Task Dashboard
- User Profile
- Responsive Design

## Technologies:
- React
- Chakra UI
- Axios
- React Router DOM
